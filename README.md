📝 Intelligence Document (AI-Native Word Clone)
Overview

This project is an AI-native word document editor.
Unlike Microsoft Word or Google Docs (built on digital foundations), this editor is built with artificial intelligence as the foundation.

Documents are alive → they suggest, critique, and adapt as you write.

The system uses multiple AI agents (grammar, style, logic, layout, etc.) to collaborate with the user.

The goal: writing becomes a conversation with the document itself, not just text on a static page.

## Backend Tests

The backend includes a Jest test suite covering document storage, plugin actions, and AI endpoints.

Run the tests with:

```bash
cd backend
npm test
```

The tests mock OpenAI calls, so no API key is required.
